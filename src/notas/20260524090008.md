# Proposições — axiomas, teoremas e outras afirmações

A confusão entre estes termos é comum porque o uso quotidiano os trata como sinónimos. Em epistemologia e lógica formal, cada um tem um papel distinto e uma relação específica com a ideia de prova.

A distinção central: **axiomas assumem-se, teoremas provam-se, conjecturas aguardam prova, hipóteses testam-se.**

## Mapa de conceitos

| Conceito | Natureza | Nota |
|---|---|---|
| [[20260524090000]] | Aceite, não provado | Ponto de partida de um sistema formal |
| [[20260524090009]] | Provado a partir de axiomas | A prova é o que o distingue |
| [[20260524090007]] | Aceite, historicamente "pedido" | Hoje equivalente a axioma |
| [[20260524090004]] | Provado, auxiliar | Serve para demonstrar outro resultado |
| [[20260524090002]] | Provado, consequência directa | O trabalho pesado já foi feito |
| [[20260524090001]] | Não provada, suspeita-se verdadeira | Um teorema em espera |
| [[20260524090003]] | Assumida para teste ou argumento | Científica: falseável; matemática: premissa local |
| [[20260524090010]] | Defendida, não provada | Domínio da argumentação, não da lógica formal |

## Hierarquia de certeza

Do mais certo ao mais incerto:

1. **[[20260524090000|Axioma]]** — aceite, não provado
2. **[[20260524090009]]** — provado a partir de axiomas
3. **[[20260524090002]]** — provado a partir de um teorema
4. **[[20260524090004]]** — provado, auxiliar de um teorema
5. **[[20260524090001]]** — não provada, suspeita-se verdadeira
6. **[[20260524090003]]** — assumida para teste ou argumento
