# Axioma

Proposição aceite sem demonstração. Não se prova — assume-se como ponto de partida.

Os **axiomas de Euclides** são o exemplo clássico: "por dois pontos passa uma única reta" não se demonstra, aceita-se. Num sistema formal, os axiomas definem o que o sistema "acredita" antes de qualquer raciocínio.

A impossibilidade de provar um axioma não é uma fraqueza — é a sua função. Todo o sistema dedutivo repousa sobre proposições não-demonstradas; se tentasses provar tudo, entrarias numa regressão infinita.

- Ao contrário do [[20260524090009]], o axioma não tem prova.
