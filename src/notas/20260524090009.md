# Teorema

Proposição demonstrada a partir de [[20260524090000|axiomas]] (e de outros teoremas já provados), usando regras de inferência.

O **teorema de Pitágoras** não se assume — prova-se. A prova é precisamente o que distingue o teorema do [[20260524090000]]: o axioma é o ponto de partida, o teorema é o ponto de chegada de um raciocínio válido.

Um teorema pode, por sua vez, servir de base para demonstrar outros teoremas. A cadeia dedutiva — axiomas → teoremas → mais teoremas — é a espinha dorsal dos sistemas formais.
